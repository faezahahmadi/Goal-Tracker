
Goal Tracker Project
A simple and interactive Goal Tracker web application built with React and Material-UI. This app allows users to create, view, and manage personal goals efficiently. Designed with modern UI components, form validation, and smooth navigation, it helps you stay organized and motivated.
Features
Create Goals: Add new goals with title, description, due date, and status.
Goal List: View all goals in a card-based layout with important details.
Goal Details: Click on a goal to see more information and edit if needed.
Responsive Design: Works on desktop and mobile devices.
Form Validation: Built with React Hook Form for smooth and validated input.
Routing: Clean navigation using React Router, including dynamic routes for goal details.
Technologies Used
React – Frontend framework
React Router – Page navigation
Material-UI (MUI) – UI components and styling
React Hook Form – Form handling and validation
JavaScript (ES6+) – Main programming language
